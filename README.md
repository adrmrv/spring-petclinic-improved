# Spring Petclinic improved

Improved with :
* Spring MVC
* JDBC
* SQLite
* Lombok
* Thymeleaf
* Bootstrap
* Webjar

## Run

```
./mvnw spring-boot:run
```
The database is included (clinic.db)
